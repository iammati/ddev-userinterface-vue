# ddev-ui

Experimental UI for ddev based on golang as backend and vue3 as frontend.

## Requirements
- go v1.17
- preconfigured ddev-local installation

## Installation
```sh
git clone https://github.com/iammati/ddev-userinterface-vue
cd ddev-userinterface-vue
ddev start
ddev pnpm i
ddev pnpm run build
./app
```
